## 📌 What this app does

github-streak shows your real GitHub streak by counting both public and private commits.

Many streak tools only look at public work. That can make your streak look lower than it is. This app uses GitHub data to give you a fuller view of your daily coding activity.

It is made for users who want a simple way to check their streak and see steady progress over time.

## ✨ Main features

- Counts public and private commits
- Shows your current streak in a clear view
- Helps you compare active days and missed days
- Uses GitHub data from your account
- Works with a clean web-based interface
- Supports GitHub profile display use
- Built for personal tracking and profile pages

## 📊 How the streak is counted

It uses GitHub contribution data and looks at:

- Public commits
- Private commits
